VStar Help



Introduction

VStar is intended to be an easy to use tool for variable star observation data analysis. Its current capabilities are:

  • Plotting a Light Curve or Standard Phase Plot from raw observation data, optionally with error bars for each observation.
  • Showing a table of the raw observations associated with the light curve.
  • Plotting a mean light curve (again, with optional error bars) along with the raw light curve.
  • Showing a table of mean observations along with the standard error of the average for each.

The plots mentioned above have magnitude on the Y axis and Julian Day (or phase) on the X axis.

When the Plot Observations and Means Mode radio button is selected, the number of days per bin size can be specified. This will change both the plot and the means table seen when in Means List mode.

VStar allows data to be loaded from one of two file formats or from the AAVSO International Database (AID).



Loading Observation Data from a File

Selecting New Star from File... from the File menu or clicking the equivalent toolbar icon opens a file dialog from which a file of one of two formats can be loaded:

  1. AAVSO download format
  2. Lines of the form: Julian Day,Magnitude,[Uncertainy],[Observer Code],[Validation Flag] or simply: Julian Day,Magnitude

where [] indicates that a field is optional. In each case, a file may contain lines of tab or comma-delimited form. The fields of the second simple format have the same meaning as in the AAVSO Download Format. VStar does not support the AAVSO Space Delimited download format.

Once loaded, you will notice that when the "List Observation" mode radio button is selected, any observation lines VStar deems to have failed validation will appear in a separate split pane. Such invalid observations will not appear on any plot.

VStar detects the file's format from its content.



Loading Observation Data from AID

In order to load observations from the AID, you will first need to have a Citizen Sky account which may be obtained for free from http://www.citizensky.org.

Selecting New Star from AAVSO Database... from the File menu or clicking the equivalent toolbar icon opens dialog that allows one of the stars from the 10 star tutorial to be selected from a drop-down menu, or a star name, unique ID, or alias to be entered. Note that currently, only the drop-down menu is enabled.

After selecting the star for which you want observations and the Julian Day range, a Citizen Sky login dialog will appear. Once authenticated, the data for the star's observations within that range are loaded, and the plot or table for each mode of the corresponding radio buttons in the Mode pane are created.

For a given run of VStar, you only need to enter your Citizen Sky credentials just once.



Phase Plots

Selecting Phase Plot... from the Analysis menu opens a dialogue where you can enter the period and epoch for a star. Values from the AAVSO International Variable Star Index (VSX) are used as the default, if they are available. VStar will then fold the data and display a Phase Plot of magnitude versus phase. Select New Phase Plot to change the period or epoch and generate a new phase plot.

Saving Plots and Tables

Any chart can be saved as a PNG file via the File menu's Save... item or the equivalent toolbar icon.

Table saving is yet to be implemented.



Printing Plots and Tables

Any chart or data table can be printed via the File menu's Print... item or the equivalent toolbar icon.



Setting Preferences

Preferences can be set via the File menu's Preferences... item or the equivalent toolbar icon.
