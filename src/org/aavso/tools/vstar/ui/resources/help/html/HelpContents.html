VStar Help



Introduction

VStar is a multi-platform, easy-to-use variable star data visualization tool.

Its current capabilities are:

  • Sourcing data from a file or the AAVSO database.
  • Plotting a Light Curve or Phase Plot from raw observation data, optionally with error bars for each observation.
  • Showing a table of the raw observations associated with the light curve.
  • Plotting a mean light curve (again, with optional error bars) along with the raw light curve.
  • Showing a table of mean observations along with the standard error of the average for each.
  • Performing period analysis.

The plots mentioned above have magnitude on the Y axis and Julian Day (or phase) on the X axis.

When the Mean Plot tab is selected, the number of days per bin size can be specified. This will change both the plot and the means table (see the Mean List tab).

VStar allows data to be loaded from one of two file formats or from the AAVSO International Database (AID).



Loading Observation Data from a File

Selecting New Star from File... from the File menu or clicking the equivalent toolbar icon opens a file dialog from which a file of one of two formats can be loaded:

  1. AAVSO download format (CSV and TSV formats only currently, not VOTable)
  2. Lines of the form: Julian Day,Magnitude,[Uncertainy],[Observer Code],[Validation Flag] or simply: Julian Day,Magnitude

where [] indicates that a field is optional. In each case, a file may contain lines of tab or comma-delimited form. The fields of the second simple format have the same meaning as in the AAVSO Download Format. VStar does not support the AAVSO Space Delimited download format.

Once loaded, you will notice that when the Observation List tab is selected, any observation lines VStar deems to have failed validation will appear in a separate split pane. Such invalid observations will not appear on any plot.

VStar detects the file's format from its content.



Loading Observation Data from AID

In order to load observations from the AID, you will first need to have a Citizen Sky account which may be obtained for free from http://www.citizensky.org.

Selecting New Star from AAVSO Database... from the File menu or clicking the equivalent toolbar icon opens dialog that allows one of the stars from the 10 star tutorial to be selected from a drop-down menu. Alternatively, a star name, unique ID, or alias may be entered.

After selecting the star for which you want to retrieve observation data and the Julian Day range (or all data; take care: this may result in a lot of data being retrieved), a Citizen Sky login dialog will appear. Once authenticated, the data for the star's observations within that range are loaded, and charts and tables for the data, including means, are created.

For a given run of VStar, you only need to enter your Citizen Sky credentials just once.



Phase Plots

Selecting Phase Plot... from the Analysis menu opens a dialogue where you can enter the period and epoch for a star. Values from the AAVSO International Variable Star Index (VSX) are used as the default, if they are available. VStar will then fold the data and display a Phase Plot of magnitude versus phase. Click the New Phase Plot button to change the period or epoch and generate a new phase plot.



Period Searches

  • Date Compensated Discrete Fourier Transform (DC DFT)
  • Phase Dispersion Minimization
  • Other possibilities: WWZ, CLEANEST, ANOVA (analysis of variance), ...

Only the first of these is currently implemented.

Selecting an item from the Analysis->Period Search sub-menu performs a period analysis on the loaded dataset. This uses the current mean source series (e.g. Visual, Johnson V) as the series on which to perform the analysis. This can be changed via the Change Series button (and the ensuing dialog) in the Mean Plot tab.

The period analysis window dialog that opens after a delay contains plots of amplitude against frequency and period, a table of all results, and a table of the top 20 hits ranked by amplitude. The latter is a good starting point for testing candidate periods. Clicking an entry in this table highlights the corresponding table entry or data-point in the other views. You can test different candidate periods by creating phase plots with those periods.



Saving Plots and Tables

Any chart can be saved as a PNG file via the File menu's Save... item or the equivalent toolbar icon.

Tables in the Observation List tab can be saved as CSV files, but Mean List table saving is yet to be implemented. For the latter, what may also be needed is to save not just the data, but also the parameters that lead to it.



Printing Plots and Tables

Any chart or data table can be printed via the File menu's Print... item or the equivalent toolbar icon.



Setting Preferences

Preferences can be set via the File menu's Preferences... item or the equivalent toolbar icon.

Currently, only plot color preferences are implemented.
